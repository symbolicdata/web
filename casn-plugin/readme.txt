== Description ==

This plugin provides Wordpress shortcodes that are used at the site
http://www.fachgruppe-computeralgebra.de to display various information
extracted from the SymbolicData RDF database at http://symbolicdata.org/Data
and also from RDF Data at the local CASN node
http://www.fachgruppe-computeralgebra.de/rdf/.

To run the examples you have additionally to install the EasyRdf-PHP-Bibliothek
http://www.easyrdf.org in a subdir lib.

For more information about the Computer Algebra Social Network CASN see
http://wiki.symbolicdata.org/CASN and also the demonstration site of the SD
project at http://symbolicdata.org/info.

== Installation and Configuration ==

- Plugin-Installation: As usually to be rolled out in the plugins directory of
  your Wordpress installation.

- Configuration: Shortcodes can be activated and passivated in 'main.php'
